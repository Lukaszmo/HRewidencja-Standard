# Timesheet
App features:
- Registration of working time
- Electronical vacation requests
- Kanban table
- Reports module

## Tech Stack:
Backend:
- PHP 7
- Symfony 4
- Api Platform
- Doctrine

Frontend:
- React JS
- Semantic UI

## Installation:
```sh
composer install
npm install
npm run build
php bin/console server:run
open http://127.0.0.1:8000
```
